--- layout: post title: 'Workaround : Ext JS 5.1.0 checkboxmodel ''deselect'' defect' date: '2015-01-11T22:17:00.002-08:00' author: Balagangadhar Addanki tags: - extjs5 modified_time: '2015-01-11T22:17:56.031-08:00' blogger_id: tag:blogger.com,1999:blog-2232263284250046405.post-5638565641706670675 blogger_orig_url: http://www.balasblog.com/2015/01/workaround-ext-js-510-checkboxmodel.html --- ExtJS 5.1.0 is having a bug in checkboxmodel. While using checkbox model, it is not firing deselect/selection change when the model is 'MULTI' which is a default value.

Use following snippet to make it work.

 selModel: {
          selType: 'checkboxmodel' 
          model : 'SIMPLE'

}
