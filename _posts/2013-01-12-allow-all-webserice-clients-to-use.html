--- layout: post title: Allow All WebSerice Clients To Use Liferay Webservice date: '2013-01-12T03:36:00.003-08:00' author: futurehelp tags: - liferay modified_time: '2013-01-12T03:42:46.595-08:00' blogger_id: tag:blogger.com,1999:blog-2232263284250046405.post-7813293597347274380 blogger_orig_url: http://www.balasblog.com/2013/01/allow-all-webserice-clients-to-use.html --- As we know, there is an entry that we need to enter into portlet-ext.properties to access webservice from the client.

axis.servlet.hosts.allowed=192.168.100.100,....

But if we want to allow every client to use these webservices created by liferay i.e. if we don't want any restriction to access liferay webservice then keep the entry blank in portlet-ext.properties.

axis.servlet.hosts.allowed=
